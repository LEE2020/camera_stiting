下面的示例说明如何从 3×3 的内参矩阵中提取焦距和主点信息。

```markdown
相机的内参矩阵通常是如下形式：
   [ fx   0  cx ]
K = [  0  fy  cy ]
   [  0   0   1 ]
其中，
   fx = K[0,0] 为横向焦距，
   fy = K[1,1] 为纵向焦距，
   cx = K[0,2] 为主点横坐标，
   cy = K[1,2] 为主点纵坐标。
例如在 Python 中：
---------------------------
import numpy as np

# 示例内参矩阵
K = np.array([[800,   0, 640],
              [  0, 800, 360],
              [  0,   0,   1]])

fx = K[0, 0]
fy = K[1, 1]
cx = K[0, 2]
cy = K[1, 2]

print("焦距：", fx, fy)
print("主点：", cx, cy)
---------------------------

这样就可以从内参矩阵中正确提取出焦距和主点信息。
```
